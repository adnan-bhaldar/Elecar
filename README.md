### Run Locally

To run **Elecar** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/adnan-bhaldar/elecar.git
```

Windows:

```bash
git clone https://github.com/adnan-bhaldar/elecar.git
```
